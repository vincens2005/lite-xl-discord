# Installation
## prerequisites:
  `make`, `gcc`, `cmake`
	the clib branch currently only supports linux.
	If you're not on linux, I recommend using [litepresence](https://github.com/TorchedSammy/litepresence)
	
## actual installation
clone this repo with `--recursive`
run `make`,
then `make install`
